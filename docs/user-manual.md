## はじめに

FluentEdge AI は、脳科学ベースの AI 英会話トレーニングアプリです。
以下の特徴で効率的な英語力向上を実現します:

- **AI による個別最適化**: Claude AI が会話相手となり、リアルタイムでフィードバック
- **忘却曲線に基づく復習**: FSRS アルゴリズムで最適なタイミングで復習を提案
- **12 種類のトレーニング**: 会話、英作文、シャドーイング、発音練習など多角的にカバー
- **弱点の自動分析**: 学習データから苦手パターンを検出し重点的に出題

### 推奨学習フロー

```text
1. ログイン -> ダッシュボードで本日の復習状況を確認
2. 復習がある場合 -> FSRS 復習で復習アイテムを消化 (5-10分)
3. AI フリートーク でシーン別の会話練習 (10-15分)
4. 瞬間英作文 で文法力を強化 (5-10問)
5. 間違えた表現は自動的に復習スケジュールに登録
6. ダッシュボードで学習の進捗を確認
```

## アカウント作成・ログイン

### 新規登録

1. アプリのトップページにアクセス
2. 「新規登録」ボタンをクリック
3. 以下の情報を入力:
   - **メールアドレス**: 有効なメールアドレス
   - **パスワード**: 8 文字以上
   - **名前**: 表示名
4. 「登録」ボタンをクリック
5. 自動的にログインしてダッシュボードに遷移

### ログイン

1. トップページで「ログイン」をクリック
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

### プロフィール設定

ログイン後、プロフィールで以下を設定できます:

| 設定項目 | 説明 | 選択肢 |
| --- | --- | --- |
| 目標レベル | 目指す英語レベル | A2, B1, B2, C1, C2 |
| 日次目標 | 1日の学習目標時間 | 5分-60分 |
| 母国語 | 母国語の設定 | ja (日本語) |

## ダッシュボード

ログイン後のホーム画面です。学習進捗を一目で確認できます。

### 表示項目

| 項目 | 説明 |
| --- | --- |
| 連続学習日数 (ストリーク) | 連続で学習した日数 |
| 累計学習時間 | これまでの総学習時間 |
| 総セッション数 | AI フリートークの実施回数 |
| 総復習完了数 | 完了した復習の累計 |
| 学習済み表現数 | 新たに学んだ表現の累計 |
| 平均文法正答率 | 文法の正確さの平均スコア |
| 直近 7 日間の日次統計 | グラフ形式で日ごとの学習量を表示 |
| 未復習アイテム数 | 本日復習が必要なアイテム数 |

### ストリークについて

- 毎日少しでも学習するとストリークが加算されます
- 1 日でも学習しない日があるとストリークはリセットされます
- ストリーク維持が学習習慣の定着に効果的です

## AI フリートーク

Claude Sonnet 4.5 AI と英語でテキスト会話を行う機能です。
ビジネスシーンに特化した 5 つのモードで実践的な会話練習ができます。

### 会話モード

| モード | 内容 | 想定シーン |
| --- | --- | --- |
| Meeting | 会議での発言・議論 | チームミーティング、報告会議 |
| Presentation | プレゼンテーション練習 | 社内発表、クライアント提案 |
| Negotiation | 交渉・折衝 | 契約交渉、条件調整 |
| Small Talk | 雑談・アイスブレイク | ランチ、出張先の雑談 |
| Interview | 面接練習 | 英語面接、海外転職 |

### 使い方

1. サイドバーから「Talk」を選択
2. 会話モードを選択
3. 必要に応じてシナリオを入力 (例: "Quarterly budget review")
4. 「開始」をクリック -> AI が最初のメッセージを送信
5. テキスト入力欄に英語でメッセージを入力して送信
6. AI 応答と同時にフィードバックパネルが表示される

### フィードバックの見方

| 項目 | 説明 |
| --- | --- |
| 文法エラー | 誤り箇所、修正案、解説 |
| 表現アップグレード | より自然な言い回しの提案 |
| 発音ノート | テキストベースの発音ヒント |
| ポジティブフィードバック | 良かった点のコメント |
| 語彙レベル | 使用語彙の CEFR レベル推定 |

### 効果的な使い方のヒント

- 1 セッション 10-15 分を目安に集中して取り組む
- フィードバックで指摘された表現を次のメッセージで意識的に使う
- シナリオ説明を具体的に書くと、より実践的な会話が展開される
- 過去のセッション履歴で振り返りが可能

## 瞬間英作文 (Flash Translation)

日本語の文を見て、制限時間内に英語に翻訳するドリル形式のエクササイズです。

### 使い方

1. メニューから「Flash」を選択
2. 問題がカード形式で表示される (デフォルト 5 問)
3. 日本語の文を読む
4. 制限時間内 (デフォルト 15 秒) に英語を入力
5. 「チェック」ボタンで回答を送信
6. スコアと解説が表示される

### スコアと復習登録

| スコア | 判定 | 復習登録 |
| --- | --- | --- |
| 0.7 以上 | 合格 | 登録されない |
| 0.7 未満 | 要復習 | 自動的に復習スケジュールに追加 |

### 難易度レベル

ユーザーの目標レベルに応じて問題が動的に生成されます:

| レベル | 内容例 |
| --- | --- |
| A2 | "この会議は 3 時に始まります" |
| B1 | "先日のメールの件について確認したいのですが" |
| B2 | "この提案は検討に値すると思いますが、いくつか懸念点があります" |
| C1 | "市場環境の変化を踏まえて、戦略の見直しを提案させていただきます" |

## FSRS 復習

FSRS (Free Spaced Repetition Scheduler) アルゴリズムによる忘却曲線ベースの間隔反復学習です。

### 使い方

1. メニューから「Review」を選択
2. 今日の復習対象アイテムが一覧表示される
3. カード形式で復習アイテムが表示される
4. 自分で回答を考える
5. カードをめくって正解を確認
6. 4 段階で自己評価する

### セルフレーティング

| ボタン | 意味 | 次回復習への影響 |
| --- | --- | --- |
| Again (1) | 完全に忘れていた | 短い間隔で再出題 |
| Hard (2) | 思い出せたが困難 | やや短い間隔 |
| Good (3) | 適度な努力で思い出せた | 標準的な間隔 |
| Easy (4) | 即座に思い出せた | 長い間隔 |

### 復習のコツ

- 毎日の復習を欠かさない (溜めると負荷が高くなる)
- 正直にレーティングする (自分に甘くすると定着しにくい)
- 朝一番の復習が記憶定着に効果的

## シャドーイング

AI が生成した英語音声を聞きながら、同時に声に出して練習する機能です。

### 使い方

1. メニューから「Shadowing」を選択
2. レベルとトピックを選択
3. 素材が生成される (テキスト + 音声)
4. 音声を再生して聞く
5. 音声に合わせて声に出して練習
6. 自分の音声を録音して評価

### 速度調整

再生速度を 0.5 倍から 2.0 倍まで調整できます:

| 速度 | 推奨 |
| --- | --- |
| 0.5x-0.75x | 初級者・細部の確認 |
| 1.0x | 通常速度 |
| 1.25x-1.5x | 上級者の挑戦 |
| 2.0x | 超上級者 |

## パターン練習

ビジネス英語でよく使われる表現パターンを繰り返し練習する機能です。

### カテゴリ

- **Meetings**: 会議で使う表現 (意見を述べる、賛成/反対する等)
- **Presentations**: プレゼンテーション表現 (導入、結論等)
- **Negotiations**: 交渉表現 (提案する、条件を出す等)
- **Emails**: ビジネスメール表現 (依頼、確認等)
- **Socializing**: 社交表現 (自己紹介、雑談等)

### 使い方

1. メニューから「Pattern」を選択
2. カテゴリを選択
3. 練習問題が表示される
4. パターンに沿って英文を作成
5. AI が評価してフィードバック

### 習熟度ステージ

| ステージ | 説明 |
| --- | --- |
| Understood (理解済み) | パターンの意味を理解した段階 |
| Drilling (反復練習中) | 繰り返し練習して定着させる段階 |
| Acquired (習得済み) | フリートークで自然に使えた段階 |

## 発音練習

日本語話者が苦手な英語の音を集中的に練習する機能です。

### 対象音素

- **L / R**: light vs right
- **TH**: think, this
- **V / B**: vest vs best
- **F**: five, phone
- **W**: water, world
- **母音**: bat vs but, sit vs seat

### 使い方

1. メニューから「Pronunciation」を選択
2. 対象音素を選択 (または自動選択)
3. 練習文が表示される
4. マイクボタンを押して音声を録音
5. Azure Speech Services が発音を評価
6. スコアと改善ポイントが表示される

## モゴモゴ英語訓練

ネイティブスピーカーの自然な発話で起きる音声変化を聞き取る練習です。

### 音声変化パターン

| パターン | 例 | 変化後 |
| --- | --- | --- |
| リンキング | "want to" | "wanna" |
| リダクション | "going to" | "gonna" |
| フラッピング | "water" | "wader" (T が D に) |
| 弱化 | "him" | "'im" |
| 脱落 | "next day" | "nex' day" |

### 使い方

1. メニューから「Mogomogo」を選択
2. 音声変化パターンの素材一覧が表示される
3. 音声を聞く
4. 聞き取った内容をテキストで入力
5. 正解と比較してフィードバック

## リスニング理解度テスト

AI が生成した英語音声を聞いて、内容理解を確認するテストです。

### 使い方

1. メニューから「Comprehension」を選択
2. レベルとトピックを選択
3. 音声が再生される
4. 質問に回答 (選択式または記述式)
5. 正答率とフィードバックが表示される

## 高度な分析

学習データから詳細な分析レポートを生成する機能です。

### 分析内容

- **スキルレーダー**: Speaking, Listening, Grammar, Vocabulary, Pronunciation の 5 軸
- **弱点パターン分析**: 苦手な文法パターンの特定
- **上達トレンド**: 週次/月次の成長曲線
- **学習推奨**: AI による次の学習内容の提案

### 週次レポート

毎週の学習結果をまとめたレポートが生成されます:

- 今週の学習時間
- 新しく習得した表現
- 改善が見られたスキル
- 来週の推奨学習プラン

### 月次レポート

月間の総合的な学習レポートです:

- 月間学習統計
- CEFR レベルの推定変化
- 長期的な成長トレンド
- コスト効率分析

## サブスクリプション管理

### プラン一覧

| プラン | 月額 | 主な機能 |
| --- | --- | --- |
| Free | 無料 | AI フリートーク (1日3回), 瞬間英作文 (1日10問), 復習 |
| Standard | 月額 980 円 | 無制限会話, 全機能利用, 週次レポート |
| Premium | 月額 1,980 円 | Standard + リアルタイム音声, 発音評価, 月次レポート |

### プラン変更手順

1. メニューから「Subscription」を選択
2. 希望のプランを選択
3. 「アップグレード」ボタンをクリック
4. Stripe の決済ページに遷移
5. クレジットカード情報を入力して決済
6. 即座にプランが反映される

### 解約手順

1. メニューから「Subscription」を選択
2. 「解約」ボタンをクリック
3. 確認ダイアログで「はい」を選択
4. 現在の期間終了まで利用可能
5. 期間終了後に Free プランに移行

## FAQ

### Q: AI の応答が返ってこない

**A**: 以下を確認してください:

- インターネット接続が安定しているか
- ブラウザのキャッシュをクリアして再度試す
- しばらく待ってから再度試す (LLM サーバーが一時的に混雑している可能性)

### Q: 復習アイテムが溜まってしまった

**A**: 毎日少しずつ消化することをお勧めします。一度に全てやろうとせず、1日 10-15 件程度を目安に取り組みましょう。時間が経つとアイテムは自然に減っていきます。

### Q: 目標レベルを変更したい

**A**: プロフィール設定から「目標レベル」を変更できます。変更すると、以降の問題生成が新しいレベルに合わせて調整されます。

### Q: データのエクスポートはできますか

**A**: 現在はデータエクスポート機能は未実装です。今後のアップデートで対応予定です。

### Q: スマートフォンで利用できますか

**A**: はい。FluentEdge AI はレスポンシブデザインに対応しており、スマートフォンのブラウザからアクセスできます。PWA マニフェストも提供しているため、ホーム画面に追加してアプリのように利用できます。

### Q: 瞬間英作文の制限時間を変更したい

**A**: 現在は制限時間の変更機能は未実装です。デフォルトの 15 秒で練習することを推奨しています。瞬間的な英作文力を鍛えるためには、時間制限が効果的です。

### Q: 支払い方法について

**A**: Stripe を通じてクレジットカード決済に対応しています。Visa, Mastercard, JCB, American Express が利用可能です。

### Q: 退会したい

**A**: サブスクリプションを解約した上で、サポートまでご連絡ください。アカウントとデータの削除を行います。
